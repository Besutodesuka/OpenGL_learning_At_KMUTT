# about this project
this is 2D art of magic orb in twilight forest using math equation to form many mathematic shape, and animate them with fragmentshader to utilize GPU capability to animate magic orb effect.

you can click any where in the screen and the orb will absorb magic power from the forest the star that orbit around will grow purple and circulate faster as power is gained.
